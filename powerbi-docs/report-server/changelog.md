# <a name="change-log-for-power-bi-report-server"></a>Power BI Report Server에 대한 변경 로그

Power BI 보고서 서버에 대한 이 변경 로그는 각 릴리스 빌드에 대한 버그 픽스와 새 항목을 나열합니다.

새로운 기능에 관한 자세한 내용은 [Power BI Report Server의 새로운 기능](whats-new.md)을 참조하세요. 

## <a name="october-2020"></a>2020년 10월
- **Power BI Report Server**
    - *버전: 1.9.7627.11028(빌드 15.0.1104.264), 릴리스 날짜: 2020년 11월 18일*
        - 버그 수정
            - 사용자가 포털을 통해 사이트 설정의 필드를 변경하지 못하게 하는 문제를 해결했습니다.
            - ‘EnterData’ 데이터 원본을 사용하는 경우 Power BI 보고서 새로 고침과 관련된 문제를 해결했습니다.
            - 향상된 데이터 세트 메타데이터를 사용한 일부 모델 새로 고침과 관련된 문제를 해결했습니다.
            - 일부 Power BI 보고서를 보고서 서버에 게시할 수 없는 문제를 해결했습니다.
    - ‘버전: 1.9.7604.41261(빌드 15.0.1104.239), 릴리스 날짜: 2020년 10월 27일’
         - 기능
            - Power BI Report Server의 향상된 데이터 세트 메타데이터를 지원하도록 설정했습니다.
            - DirectQuery 및 새로 고침을 위해 Power BI 보고서에 대한 연결을 업데이트하는 기능을 사용하도록 설정했습니다(자세한 내용은 [데이터 원본 연결 문자열 변경](./connect-data-source-apis.md) 참조).
        - 보안 업데이트
        - 버그 수정
            - 사용자가 Power BI 보고서 새로 고침 일정을 변경할 수 없도록 하는 문제를 해결했습니다.
            - 자격 증명이 만료된 보고서를 관리할 때 사용자에게 표시되는 혼란스러운 오류 메시지를 수정했습니다.
            - 이름에 마침표를 사용하여 보고서를 내보낼 때 발생하는 문제를 해결했습니다.
            - 테이블릭스의 화면 읽기 프로그램 문제를 해결했습니다.
            - 일부 환경에서 로그 파일이 비어 있게 되는 문제를 해결했습니다.
            - “Power BI에 연결” 대화 상자가 닫히지 않는 문제를 해결했습니다.
            - 최신 HTML DOCTYPE을 사용하도록 MHTML 렌더러를 업데이트했습니다.

- **Power BI Desktop(Power BI Report Server에 최적화됨)**
   - ‘버전: 2.86.961.0(2020년 10월), 릴리스 날짜: 2020년 10월 27일’(새 빌드 및 새 버전)
        - Power BI Report Server 연결에 필요한 변경 내용 포함(2020년 10월)

            - 폴더 및 보고서 이름에서 작은따옴표와 관련된 두 가지 문제를 해결했습니다.
            - 가로 스크롤을 특정 브라우저 및 레코드 참조 기능과 연결하는 문제를 해결했습니다.
            - 보고서를 여는 동안 예약된 새로 고침에서 기본 모델의 스키마 오류가 발생할 수 있는 문제를 해결했습니다.

            - LoadReport를 실행하는 사용자 지정 애플리케이션에서 TrustedHeader 오류가 잘못 수신되는 문제를 해결했습니다.
            - 예약된 새로 고침에서 부하가 많은 경우 새로 고침이 실패하는 문제를 해결했습니다.
            - 보고서 이름이 폴더 이름과 일치하는 경우 보고서가 잘못된 위치에 저장하는 문제를 해결했습니다.
            - 문서 구조에서 탭 이동 문제를 해결했습니다.
            - DAX 쿼리를 사용할 때 발생하는 데이터 기반 구독 실패 문제를 해결했습니다.
            - URL 액세스에서 FindString이 일치하는 항목을 찾지 못하는 문제를 해결했습니다.
            - 보고서를 이동할 때 포함된 데이터 원본이 중단되는 문제를 해결했습니다.
            - 특정 데이터 원본에 대해 예약된 새로 고침이 실패하는 문제를 해결했습니다.
            - 잘못된 요청이 발생하는 경우를 줄이기 위해 보고서 일정에 대한 유효성 검사를 추가했습니다.
