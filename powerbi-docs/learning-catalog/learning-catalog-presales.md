# <a name="partner-pre-sales-learning-catalog"></a>파트너 사전 판매 학습 카탈로그

고객에게 주요 기능을 시연해야 합니까? 

다음 카탈로그에는 핵심 정보에서 특정 도메인까지, 가장 기본적인 것에서 가장 고급 기능까지 다양한 내용이 정리되어 있습니다. 콘텐츠가 여러 형식으로 제공되는 경우 알려 드리며, 가장 적합한 교육 형식을 선택하면 됩니다.
