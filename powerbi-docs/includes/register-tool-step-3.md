3. ‘1단계 - Power BI에 로그인’에서 Power BI 테넌트에 속한 사용자로 로그인합니다. Azure AD 앱이 이 사용자로 등록됩니다.

    이미 로그인한 경우 Azure AD 앱을 만드는 데 사용하려는 사용자로 로그인했는지 확인합니다. 사용자를 변경하려면 ‘로그아웃’ 링크를 선택하고 도구가 다시 시작되면 다시 로그인합니다.
