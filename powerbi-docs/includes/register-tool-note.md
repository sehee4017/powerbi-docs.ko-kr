>선택적 단계를 건너뛴 경우에도 샘플 Power BI 앱을 다운로드할 수 있습니다. 그러나 다운로드한 앱의 코드에는 등록하는 동안 입력하지 않은 속성이 없습니다. 예를 들어 작업 영역을 만들지 않은 경우 샘플 앱에는 ‘작업 영역 ID’가 포함되지 않습니다.
